## How to Install This Node
1. move to `.node-red/`
2. remove old version
     - `npm uninstall "@eric-w/node-red-dinkle-modbus"`
3. reinstall latest module from github
     - `npm install git+https://github.com/Xhadow0823/node-red-dinkle-modbus.git`
